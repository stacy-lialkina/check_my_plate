# Настройка OpenAI API

## Быстрый старт

1. **Получите API ключ:**
   - Перейдите на https://platform.openai.com/api-keys
   - Войдите или создайте аккаунт
   - Нажмите "Create new secret key"
   - Скопируйте ключ (он показывается только один раз!)

2. **Установите ключ как переменную окружения:**

   **macOS/Linux:**
   ```bash
   export OPENAI_API_KEY="sk-..."
   ```

   **Windows (PowerShell):**
   ```powershell
   $env:OPENAI_API_KEY="sk-..."
   ```

   **Windows (CMD):**
   ```cmd
   set OPENAI_API_KEY=sk-...
   ```

3. **Запустите приложение:**
   ```bash
   python3 app.py
   ```

## Постоянная настройка (опционально)

Чтобы не вводить ключ каждый раз, добавьте в `~/.bashrc` или `~/.zshrc`:
```bash
export OPENAI_API_KEY="sk-..."
```

Затем выполните:
```bash
source ~/.bashrc  # или source ~/.zshrc
```

## Проверка

После запуска приложения вы должны увидеть:
- Если ключ установлен: приложение работает без предупреждений
- Если ключ не установлен: появится предупреждение "Warning: OPENAI_API_KEY not set. Food recognition will not work."

## Стоимость

OpenAI Vision API (GPT-4o) платный:
- Примерно $0.01-0.05 за анализ одного изображения
- Первые $5 бесплатно при регистрации

## Альтернативы

Если не хотите использовать OpenAI, можно:
- Оставить `OPENAI_API_KEY` пустым - приложение будет работать, но без автоматического распознавания
- Выбирать продукты вручную из списка
